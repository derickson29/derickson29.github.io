Projects

  • Python Project for NetApp

    While working for Ennovar contracted out to NetApp we were tasked to create a Python program to run off a bootable USB using Docker that would collect the support bundles on their machines and parse the logs using regular expressions to look for specific hardware failure signatures. The idea was that when a malfunctioning piece of hardware was returned this tool could be used to quickly identify which pieces of hardware had failed.
